# Dofas Kitchen - Online Food Delivery Platform

Welcome to Dofas Kitchen, your one-stop solution for ordering delicious meals from our restaurant delivered straight to your doorstep.


## Features

- **Search Dishes:** Find your favorite dishes or discover new ones easily.
- **Order Tracking:** Real-time tracking of your order from preparation to delivery.
- **Secure Payments:** Multiple payment options with integrated payment gateways.
- **Reviews and Ratings:** Read reviews and ratings to make informed decisions.
- **User Profiles:** Manage your orders, addresses, and personal information.
- **Responsive Design:** Enjoy the same great experience on mobile and desktop devices.

## Getting Started

### Prerequisites

- HTML
- CSS
- JavaScript

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/foodiez/food-delivery.git
   ```

2. Navigate to the project directory:
   ```sh
   cd food-delivery
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Set up environment variables:
   - Rename `.env.example` to `.env`
   - Fill in required values like database connection and API keys

5. Start the application:
   ```sh
   npm start
   ```

## Usage

1. Sign up or log in to your Dp account.
2. Browse restaurants and explore their menus.
3. Add items to your cart and proceed to checkout.
4. Choose your delivery address and payment method.
5. Track your order's status in real-time.
6. Once delivered, rate and review your experience.

## Technologies Used

- Frontend: HTML, CSS, JavaScript, React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment: Stripe API
- Map: Mapbox API
